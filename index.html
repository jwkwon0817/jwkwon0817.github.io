간단한 숫자 야구 게임

설명 문제 원하는 기능 규칙 팀원

설명

주제: Java 언어를 사용하여 간단한 숫자 야구 게임 만들기
  • Java 스터디이기 때문에 Java 언어 사용
  • 이전 스터디 내용인 변수, 연산자, 조건문과 반복문 사용과 이번 스터디 내용인 메소드 선언과 호출 같은 내용을 참고하여 구성

    • 문제

    게임이 끝난 후 재시작 문구(Y/N)으로 재시작 여부 확인 후 재시작 혹은 게임 종료 기능 추가에 문제가 많았고 이를 해결

    원하는 기능

    추가하고 싶은 기능으로는 사용자가 입력했을 때 일어날 더 많은 오류를 해결하는 것(첫 입력에 숫자를 0~9가 아닌 경우 입력 시 자동으로 다시 수를 입력받을 수 있게 하는 기능)

    규칙

    • 1. 숫자는 띄어쓰기로 구분한다.
    • 2. 숫자가 같고 위치까지 같으면 스트라이크
    • 3. 숫자가 같지만 위치는 다르다면 볼
    • 4. 숫자가 다르다면 아웃
